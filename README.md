# Future Jobs Market Analysis Dashboard
This interactive Power BI dashboard provides insights into the future job market, including:
- Average Salary
- Total Job Roles
- Remote Jobs & Company Size
- Industry & Location demand

## Files
- `Future_Jobs_Dashboard.pbix` - Power BI dashboard file
- `future_jobs_dataset.csv` - Sample dataset used

## Features
- Interactive KPI cards
- Filter by location, remote option, and company size
- Visuals include bar charts, donut charts, and horizontal bars
